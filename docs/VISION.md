# Marketplace Bridge

Marketplace Bridge — это WordPress-плагин, который делает WooCommerce центральной системой управления товарами.

Источник данных:

- Ozon
- Wildberries

Все товары импортируются в WooCommerce.

WooCommerce является единственным местом редактирования.

Изменения синхронизируются обратно на маркетплейсы.

Цель проекта — не просто импортировать товары, а предоставить единый центр управления несколькими маркетплейсами.
